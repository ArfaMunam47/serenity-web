# ✨ About

Serenity is a premium relaxation landing page crafted to create a calming and immersive user experience. Every animation, interaction, and visual element has been carefully designed to encourage mindfulness and help users slow down in a fast-paced digital world.

The project demonstrates how modern UI/UX principles can be achieved using only **HTML5**, **CSS3**, and **Vanilla JavaScript**—without relying on external frameworks.
